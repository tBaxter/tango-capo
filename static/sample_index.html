Hello, world!

This is a template for a simple marketing or informational website. It includes a large callout called the hero unit and three supporting pieces of content. Use it as a starting point to create something more unique.

Easy columns

Content can be gridded off into one to six columns simply by added a 'columned' class to the parent. The children can be any element at all: divs, list items, sections, whatever. It doesn't matter.... foo class="columned"

Controllable...

By default, content will be display in three columns. However, you can set the number by adding a class specifying the number of columns: 'four-col', 'two-col', 'five-col', etc.... foo class="columned four-col"

Or automatic...

There is experimental support for 'columned auto' which should set the number of columns according to available sensible width. In fact, this column grid is set to auto width: foo class="columned auto"

Responsive.

On small screens, multiple columns will be reset to a more appropriate width. For example, four-col will collapse down to two-col on very small screens.

And layout-ready

There is also experimental support for 'columned phi' and 'columned phi reverse', which both automatically break content into two columns based on the golden ratio/golden mean.
